    \item [02:00-02:30] \emph{(30-min coffee break)}

    \item [03:40-03:50] \emph{(10min)} Creating a CernVM-FS repository (birds-eye view)
    \item [03:50-04:00] \emph{(10min)} Q\&A + closing remarks
